# nodejs-secured-api

## Setup project dependencies

Please open a terminal and run the following command:

```bash
yarn install
```

## Run project

Please open a terminal and run the following command:

```bash
yarn dev
```

## Lint the project

Please open a terminal and run the following command:

```bash
yarn lint
```
